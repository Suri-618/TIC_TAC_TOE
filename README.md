# 🎮 TIC TAC TOE (C++)

A simple console-based **Tic Tac Toe game** implemented in **C++**.  
This project demonstrates core programming concepts such as loops, conditionals, functions, arrays, and game logic.

---

## 🚀 Features

✔ Two-player mode (Player X vs Player O)  
✔ Input validation  
✔ Win condition checking  
✔ Draw detection  
✔ Clean console board display  
✔ Replay support (if implemented)

---

## 🛠 Tech Stack

- **Language:** C++
- **Compiler:** g++ / MinGW / Any C++ Compiler
- **Platform:** Console / Terminal Application

---

## 🎯 Game Rules

- The game is played on a 3x3 grid.
- Player 1 uses **X**
- Player 2 uses **O**
- Players take turns choosing positions (1–9).
- The first player to align 3 symbols horizontally, vertically, or diagonally wins.
- If all positions are filled and no one wins, the game ends in a **Draw**.
